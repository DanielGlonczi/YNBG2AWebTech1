For-For Ciklus Kódja

JavaScript Kód - For-For Ciklus:

// Egymásba ágyazott tömbök végigmenete for-for ciklus segítségével
for (let i = 0; i < xyObj.cars.length; i++) {
  const carBrand = xyObj.cars[i];
  console.log("Márkacímsor: " + (i + 1) + ". " + carBrand.name);
  
  // Belső ciklus - Modellek végigmenete
  for (let j = 0; j < carBrand.models.length; j++) {
    const model = carBrand.models[j];
    console.log("  └─ " + model);
  }
}

Magyarázat:

Külső for ciklus (i): Végigmegy a márkákon (cars tömb)

Belső for ciklus (j): Végigmegy az egyes márkák modelljeien (models tömb)

Eredmény: Teljes lista az összes márkáról és modelljéről

Információ: Ez az oldal az egymásba ágyazott tömbök feldolgozását mutatja be for-for ciklus segítségével.

Adatok forrása: Az XY személyes objektum, amely autó márkákat és modelleket tartalmaz.
