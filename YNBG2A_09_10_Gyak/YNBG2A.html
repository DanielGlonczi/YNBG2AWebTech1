A legfontosabb GitHub parancsok:

  • git clone: Repository klónozása
  • git add: Fájlok hozzáadása a staging area-hoz
  • git commit: Változtatások elkötelezése
  • git push: Változtatások feltöltése a távoli repository-ba
  • git pull: Változtatások lehúzása a távoli repository-ból
  • git status: Aktuális állapot megtekintése
  • git branch: Branch-ek kezelése
  • git merge: Branch-ek egyesítése
  • git log: Commit előzmények megtekintése
  • git checkout: Branch váltás vagy fájl visszaállítás
  • git remote: Távoli repository-k kezelése
  • git fetch: Változtatások letöltése a távoli repository-ból
  • git reset: Változtatások visszavonása
  • git stash: Változtatások ideiglenes elmentése
